# Tsy Demo

用于快速展示方言同声传译 AI 助手界面的独立 Demo，不区分前后端目录。

## 本地启动

```bash
npm install
cp .env.example .env.local
# 在 .env.local 中填写 DASHSCOPE_API_KEY
npm run dev
```

## 生产构建

```bash
npm run build
```

主要页面代码位于 `src/App.vue`，全局样式位于 `src/styles.css`。

## 演示流程

1. 点击首页麦克风或“新建会话”开始实时同传。
2. 浏览器申请麦克风权限，将单声道 16kHz PCM 音频流发送给阿里云 Fun-ASR 实时识别。
3. 每句话结束后根据音量、过零率和频谱亮度进行双人声学聚类，区分办事群众和窗口人员；判断错误时可点击交换按钮修正。
4. 最终识别文本会异步转换为普通话，并生成 AI 回复建议和实时摘要。
5. 可暂停或继续收音、切换方言、关闭译文、刷新或采用 AI 建议。
6. 点击“结束”归档会话，之后可从左侧最近会话重新查看。

会话和设置保存在浏览器本地。方言改写和语音播放通过同目录下的轻量 Node 服务调用阿里云，设置中心可一键重置演示数据。

## 阿里云能力

Demo 已接入以下阿里云百炼能力：

- 阿里云百炼 DashScope 网关；
- `cosyvoice-v3-flash` 语音合成模型；
- 阿里云官方方言音色；
- `fun-asr-realtime` 麦克风实时语音识别；
- 单麦克风双人声学聚类分离；
- 工作人员声纹录入与跨会话身份匹配（本地 MFCC 声学指纹）；
- `qwen-plus` 方言口语改写；
- 北京地域和对应 Workspace 配置。

当前音色映射：

- 粤语：龙嘉怡 `longjiayi_v3`
- 闽南语：龙安闽 `longanmin_v3`
- 东北话：龙老铁 `longlaotie_v3`
- 陕西话：龙陕哥 `longshange_v3`
- 四川话、河南话、山东话、湖南话、安徽话：龙安欢 `longanhuan_v3` 配合对应方言指令

API Key 只从 `.env.local` 或进程环境变量读取，不会发送给浏览器。配置完成后：

1. 对话卡片的播放按钮通过 WebSocket 使用阿里云 CosyVoice 流式合成；
2. 采用 AI 建议时先由 Qwen 改写为当前方言；
3. 再使用当前方言对应的官方音色自动播报；
4. 流式合成不可用时降级到完整音频接口，阿里云不可用时再降级到浏览器本地语音。

可通过设置中心查看阿里云连接状态，或访问 `/api/aliyun/config` 检查配置，但接口不会返回 API Key。

## 工作人员声纹

设置中心可在取得本人授权后录入工作人员声纹。每次录制 8 秒，建议在真实工作环境中录入 3 个样本。系统仅在 `.voiceprint-data/voiceprints.json` 保存说话人向量，不保存原始录音，该目录不会提交到 Git。

实时识别形成完整语音段后，系统优先与工作人员声纹库匹配；匹配成功显示登记姓名，未匹配时继续使用单次会话的双人声学聚类。声纹结果只用于辅助角色区分，不用于登录、审批或身份认证。
